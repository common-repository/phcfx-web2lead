== Description ==

#### Never loose a deal again!

*Wordpress2Lead for PHC CRM FX* helps you to build a contact form that you can use in your business website.

When a user sends out information through the form, it will be stored directly into your PHC CRM FX as a lead contact.
This way you know that potencial deals will have the proper attention and your lead won't be lost in your inbox!

##### NOTE: minimal PHC FX version: v15.2

== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory or install via the Add New Plugin menu;
2. Activate the plugin through the 'Plugins' menu in WordPress;
3. Enter your PHC FX backend info in 'Settings' page. Its required to set the PHC FX backend URL and your generated user's API Key;

== Frequently Asked Questions ==

= What version of my PHC FX instalation is required to work with this plugin? =
You need to have PHC FX v15.2 or higher on your installation.

= No leads are being stored in my PHC FX database. No errors shows up after sending?! What is going on? =
Possibly you're using more than one database in your PHC FX backend. If this is the case, configure the plugin's 'Database Name'
in settings to the name of the database you want to store your leads data.
